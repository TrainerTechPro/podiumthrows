Performance & Accessibility

Performance
  • CSS mask composites on GPU — no layout/paint thrash
  • Next.js <Image> with priority + sizes for both images
  • useMotionValue (not useState) — no React re-renders on mouse move
  • requestAnimationFrame throttle on velocity calculation
  • prefers-reduced-motion: disable springs, show static reveal
Accessibility
  • Both images have alt text (base image is decorative aria-hidden, reveal has descriptive alt)
  • prefers-reduced-motion: skip all animation, show athlete image statically
  • Text overlay maintains WCAG AA contrast via gradient overlay
  • CTA button remains fully focusable and keyboard-accessible
  • Gyro permission: only requested on user tap, never auto-prompted
