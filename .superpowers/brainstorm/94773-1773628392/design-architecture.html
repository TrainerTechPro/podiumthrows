Spring Physics Parameters

Framer Motion Configuration
// Cursor position — springy follow for organic feel
cursorSpring: { stiffness: 150, damping: 15, mass: 0.5 }
→ Smooth lag behind cursor, slight overshoot = "liquid" motion
// Dynamic radius — reacts to velocity
radiusSpring: { stiffness: 100, damping: 20, mass: 0.8 }
→ Slower spring = radius swells and shrinks organically
// Parallax — subtle depth on both images
parallaxSpring: { stiffness: 50, damping: 30 }
→ Base image shifts ~10px, reveal image shifts ~20px (opposite) = depth

Platform Behavior

🖱️ Desktop
Mouse cursor drives mask position. Dynamic radius from velocity. Parallax on both layers.
📱 Mobile (Gyro)
DeviceOrientation API drives mask. Tilting phone moves the reveal. Permission prompt on iOS.
📱 Fallback
Auto-animated loop: mask drifts in a slow figure-8 path. No interaction needed.
