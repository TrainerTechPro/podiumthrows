Self-Programming Wizard — Step Flow

The adaptive wizard steps. Grayed steps are conditional — they only appear based on earlier answers.

1
Program Type
Throws Only / Throws + Lifting / Lifting Only
2
Event Selection
Shot Put / Discus / Hammer / Javelin — with gender for comp weight lookup
3
Experience & PRs
Years throwing, competition level, current PR (used for implement table lookup and volume targets)
4
Available Implements
Checklist of implement weights they own — pre-populated from event/gender implement table. Athlete checks what they have.
5
Athlete Typing Conditional
Only if no existing ThrowsTyping data. Asks 3-4 key questions: adaptation speed, transfer type, recovery profile. Uses sensible defaults if skipped.
6
Training Schedule
Days per week (3-6), sessions per day (1 or 2), preferred training days, start date
7
Competition Dates Optional
Add upcoming meets to reverse-engineer peaking phases. Skip for rolling mesocycles.
8
Goals & Mode
Primary goal (improve distance, technique, consistency). Autopilot vs. Guided with Preferences mode selection.
9
Exercise Preferences Guided mode only
Preferred lifts, exercises to avoid (injuries/preference), favorite drills. Only shown if "Guided with Preferences" selected.
Review & Generate
Summary of all inputs. Preview of program structure (phases, weekly split). Confirm to generate. Slide-to-confirm on mobile.
