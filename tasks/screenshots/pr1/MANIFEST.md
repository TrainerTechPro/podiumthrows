## Themes

| Name    | Cookie value   | Visual state          |
| ------- | -------------- | --------------------- |
| `light` | `theme=light`  | `dark` class removed  |

## Filename convention

`<surface-slug>__<theme>__<viewport>.png`

Examples:

- `coach-dashboard__dark__desktop-1440.png`
- `athlete-log-session__light__mobile-390.png`
